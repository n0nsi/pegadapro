# Pegada Pro

Landing page oficial da marca **Pegada Pro**, focada em performance e proteção para atletas de Jiu-Jitsu.

Site desenvolvido com HTML, CSS e JavaScript puro, com foco em performance, responsividade e experiência de conversão.

---

## Acesso ao projeto

- 🔗 Cloudflare Pages: https://pegadapro.help4nes.workers.dev
- 🔗 Domínio oficial: https://pegadapro.com.br

---

## Sobre o projeto

A Pegada Pro é uma landing page moderna criada para apresentar e vender produtos voltados para atletas de Jiu-Jitsu, com foco em:

- Performance visual forte
- Identidade de marca esportiva
- Conversão para compra (CTA direto)
- Integração com Instagram e WhatsApp
- Estrutura leve e rápida

---

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla)
- Git & GitHub
- Cloudflare Pages (deploy)

---

## Funcionalidades

- Navbar fixa com navegação suave
- Hero section com identidade da marca
- Seção de produto com link direto de compra
- Integração com Instagram embeds
- Seção de atletas renderizada via JavaScript
- Layout responsivo (mobile-first)
- Botões de contato via WhatsApp
